    Hints

    Hint shortcode can be used as hint/alerts/notification block. There are 3 colors to choose: info, warning and danger.

**Markdown content**

stringit, frustra Saturnius uteroque inter! Oculis non ritibus Telethusa
{{< /hint >}}
